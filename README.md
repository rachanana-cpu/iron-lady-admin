\# Iron Lady Program Operations Assistant



\## Problem Statement

The internal team at Iron Lady spends significant time managing program information manually, which can lead to inefficiency and data inconsistency.



\## Solution

A lightweight internal web application that automates program management and provides an easy-to-use interface for creating, viewing, updating, and deleting programs.



\## AI Usage

AI logic is used to assist the team by generating meaningful program descriptions automatically, helping reduce repetitive manual work.



\## Technologies Used

\- HTML

\- CSS

\- JavaScript

\- Browser LocalStorage

\- AI-assisted logic



\## How to Run

1\. Clone or download the repository  

2\. Open the project directory  

3\. Run `index.html` in a web browser
